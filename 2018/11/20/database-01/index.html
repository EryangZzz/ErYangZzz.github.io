数据库结构化查询语言

SQL: Structured Query Language - 结构化查询语言 - 不区分大小写

  • DDL: Data Definition Language - 数据定义语言 - create(创建) / drop(删除) / alter(修改)
  • DML：Data Manipulation Language - 数据操作语言 - insert/delete/update(更新修改)/select
  • DCL: Data Control Language - 数据控制语言 - grant(授权)/revoke(召回权限)/commit/rollback
